# EMS 售前调研工具用户手册

## 1. 系统概述
EMS 售前调研工具是一款专为售前工程师设计的智能化调研管理系统，用于收集、分析和管理客户的能源管理需求，生成专业的调研报告，辅助售前工程师快速响应客户需求，提高工作效率和专业性。

### 1.1 主要功能
- **调研表单管理**：创建、编辑、填写和管理调研表单
- **调研模板管理**：维护各行业的调研问卷模板
- **调研报告生成**：自动生成结构化的调研报告
- **用户管理**：管理系统用户账号和权限
- **字典管理**：维护系统基础数据字典
- **产品能力管理**：管理公司产品和服务能力
- **系统日志**：查看系统操作日志
- **消息中心**：接收系统通知和报告生成消息

### 1.2 适用对象
- **超级管理员**：负责系统的整体管理和维护
- **售前工程师**：负责创建调研表单、管理模板、查看报告
- **客户用户**：负责填写分配给自己的调研表单

## 3. 仪表盘
仪表盘是系统的首页，展示了系统的关键数据和统计分析结果。

### 3.1 数据概览
- **调研客户数量**：展示系统中调研的客户总数
- **调研项目数量**：展示系统中创建的调研项目总数
- **报告生成数量**：展示系统中生成的调研报告总数

### 3.2 统计分析
- **行业分布**：通过饼图展示调研项目的行业分布情况
- **区域分布**：通过饼图展示调研项目的区域分布情况
- **客户需求排行**：通过柱状图展示客户需求的排行榜
- **客户价值图**：通过散点图展示客户价值评估结果

### 3.3 时间范围选择
可以通过页面顶部的时间范围选择器，查看不同时间范围内的数据：
- **本月**：查看本月的数据
- **本年**：查看本年的数据
- **累计**：查看所有历史数据

## 4. 调研表单管理
调研表单管理是系统的核心功能，用于创建、编辑、填写和管理调研表单。

### 4.1 创建调研表单
1. 点击左侧导航栏中的"调研表单管理"
2. 点击页面右上角的"新建表单"按钮
3. 填写表单基本信息：
   - 项目名称：调研项目的名称
   - 客户名称：客户公司的名称
   - 行业：客户所属的行业
   - 区域：客户所在的区域
4. 选择合适的调研模板
5. 点击"保存"按钮，进入表单填写页面

### 4.2 填写调研表单
1. 在调研表单列表中找到需要填写的表单
2. 点击"填写"按钮，进入填写页面
3. 根据模板要求填写调研内容
4. 点击"保存草稿"按钮，保存当前填写的内容
5. 填写完成后，点击"提交"按钮，提交表单

### 4.3 查看调研报告
1. 客户提交表单后，系统自动生成调研报告
2. 在调研表单列表中找到已提交的表单
3. 点击"查看报告"按钮，查看生成的调研报告
4. 点击"下载"按钮，下载报告为Word或PDF格式

### 4.4 管理调研表单
- **编辑**：修改表单的基本信息
- **删除**：删除不需要的表单
- **重填**：将已提交的表单重新设置为草稿状态，允许重新填写

## 5. 调研模板管理
调研模板管理用于维护各行业的调研问卷模板，为调研表单提供标准化的结构。

### 5.1 创建调研模板
1. 点击左侧导航栏中的"调研模板管理"
2. 点击页面右上角的"新增模板"按钮
3. 填写模板信息：
   - 模板名称：模板的名称
   - 行业：模板适用的行业
4. 添加章节和字段：
   - 点击"添加章节"按钮，添加调研章节
   - 点击"添加字段"按钮，为章节添加调研字段
   - 配置字段属性：字段名称、类型、是否必填、选项等
5. 点击"保存"按钮，保存模板

### 5.2 管理调研模板
- **查看**：查看模板的详细信息和结构
- **编辑**：修改模板的名称、行业和字段配置
- **复制**：复制现有模板，快速创建新模板
- **删除**：删除不需要的模板

## 6. 用户管理
用户管理用于管理系统用户账号和权限，确保系统的安全访问。

### 6.1 创建用户
1. 点击左侧导航栏中的"用户管理"
2. 点击页面右上角的"新增用户"按钮
3. 填写用户信息：
   - 姓名：用户的真实姓名
   - 用户名：登录系统的用户名
   - 类型：用户类型（内部用户或外部客户）
   - 角色：用户的角色（超级管理员、售前工程师、客户用户）
   - 客户：如果是外部客户，填写所属客户公司名称
   - 状态：用户状态（启用或禁用）
4. 点击"保存"按钮，创建用户

### 6.2 管理用户
- **编辑**：修改用户的基本信息和权限
- **启用/禁用**：启用或禁用用户账号
- **删除**：删除不需要的用户账号

### 6.3 角色管理
1. 点击左侧导航栏中的"角色管理"
2. 查看系统预设的角色和权限
3. 点击"编辑"按钮，修改角色的权限配置
4. 点击"新增角色"按钮，创建新的角色

## 7. 字典管理
字典管理用于维护系统的基础数据字典，为系统提供标准化的数据选项。

### 7.1 字典类型管理
1. 点击左侧导航栏中的"字典管理"
2. 在左侧树状菜单中选择"字典类型管理"
3. 点击"新增类型"按钮，创建新的字典类型：
   - 类型名称：字典类型的名称
   - 类型代码：字典类型的唯一标识
   - 描述：字典类型的描述
4. 点击"保存"按钮，保存字典类型

### 7.2 字典项管理
1. 在左侧树状菜单中选择一个字典类型
2. 点击"新增项"按钮，为字典类型添加字典项：
   - 显示名称：字典项的显示名称
   - 枚举值：字典项的实际值
   - 排序：字典项的排序顺序
3. 点击"保存"按钮，保存字典项

### 7.3 区域管理
1. 在左侧树状菜单中选择"区域管理"
2. 查看系统预设的中国行政区划数据
3. 点击"新增区域"按钮，添加新的区域数据：
   - 区域名称：区域的名称
   - 区域编码：区域的唯一编码
   - 父级区域：所属的父级区域
   - 区域层级：区域的层级（国家、大区、省份、城市）
4. 点击"保存"按钮，保存区域数据

## 8. 产品能力管理
产品能力管理用于管理公司的产品和服务能力，为解决方案推荐提供数据支持。

### 8.1 创建产品能力
1. 点击左侧导航栏中的"产品能力管理"
2. 点击页面右上角的"新增产品能力"按钮
3. 填写产品能力信息：
   - 产品名称：产品或服务的名称
   - 产品类型：产品类型（软件、硬件、咨询）
   - 适用行业：产品适用的行业（可多选）
   - 应用场景：产品的应用场景（可多选）
   - 描述：产品的详细描述
4. 点击"保存"按钮，保存产品能力

### 8.2 管理产品能力
- **查看**：查看产品能力的详细信息
- **编辑**：修改产品能力的信息
- **删除**：删除不需要的产品能力

## 9. 系统日志
系统日志用于查看系统的操作记录，便于审计和排查问题。

### 9.1 查看系统日志
1. 点击左侧导航栏中的"系统日志"
2. 在日志列表中查看系统的操作记录
3. 使用顶部的筛选器，按类型筛选日志：
   - 登录日志：用户登录系统的记录
   - 调研日志：调研表单相关的操作记录
   - 用户日志：用户管理相关的操作记录
   - 系统日志：系统级别的操作记录

### 9.2 日志详情
点击日志列表中的记录，查看日志的详细信息：
- 操作时间：操作发生的时间
- 操作者：执行操作的用户
- 操作类型：操作的类型
- 操作内容：操作的详细内容
- 操作结果：操作的结果（成功或失败）
- IP地址：操作者的IP地址

## 10. 消息中心
消息中心用于接收系统的通知和报告生成消息，及时了解系统的重要事件。

### 10.1 查看消息
1. 点击左侧导航栏中的"消息中心"
2. 在消息列表中查看系统发送的消息
3. 未读消息会以蓝色背景标识

### 10.2 管理消息
- **标记已读**：点击消息，将消息标记为已读
- **标记全部已读**：点击页面右上角的"全部标记为已读"按钮，将所有消息标记为已读
- **删除消息**：点击消息右侧的删除按钮，删除消息

### 10.3 消息类型
- **系统通知**：系统级别的通知，如系统升级、维护等
- **报告通知**：调研报告生成的通知，包含报告的链接

## 11. 常见问题

### 11.1 登录问题
- **忘记密码**：联系系统管理员重置密码
- **账号被禁用**：联系系统管理员启用账号
- **登录失败**：检查用户名和密码是否正确，确保网络连接正常

### 11.2 调研表单问题
- **无法创建表单**：确保已选择正确的模板，填写了所有必填信息
- **无法提交表单**：确保填写了所有必填字段，检查网络连接
- **报告生成失败**：联系系统管理员检查报告生成服务

### 11.3 模板问题
- **模板不适用**：创建新的模板或修改现有模板
- **字段配置错误**：编辑模板，修改字段配置

### 11.4 系统问题
- **页面加载缓慢**：检查网络连接，刷新页面
- **功能不可用**：联系系统管理员检查系统状态
- **数据丢失**：联系系统管理员进行数据恢复
